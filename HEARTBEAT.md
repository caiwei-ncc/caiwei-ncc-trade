# HEARTBEAT.md

## 周期任务

### 每周五（交易周结束后）
- [ ] 读本周所有 `memory/YYYY-MM-DD.md` 文件
- [ ] 更新 `MEMORY.md` 的"周期复盘记录"表格（盈亏、净值、备注）
- [ ] 检查账户净值，更新 `MEMORY.md` 中的账户状态
- [ ] 如有重大教训或规则变更，同步更新 `MEMORY.md` 核心教训部分
- [ ] 检查 `tradingview/chan-definitions.md` 与 `TRADINGVIEW.md` 是否需要更新（本周是否有新教训、新规则、定义补充）

### 每日（有交易的日子）
- [ ] 交易结束后将结果写入 `memory/YYYY-MM-DD.md`

### 每次 Heartbeat 必查
- [ ] 检查今天的 `memory/YYYY-MM-DD.md` 是否存在，不存在且今天有过对话则立刻创建并补记
- [ ] 检查 `MEMORY.md` 是否需要更新（重大事件、新规则、账户变动）
