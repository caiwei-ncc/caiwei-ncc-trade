# 分析审计清单

> 基于 `TRADINGVIEW.md`（§零~§十 + 附录）与 `tradingview/chan-definitions.md`（34 章节最终版）对齐。
> 用此清单验证 AI 是否严格按缠论推导。缺 3 条以上说明分析不够严格。
> 最后更新：2026-03-27

---

## 一、交易过滤器

- [ ] **防狼术**（§十七）：4H MACD DIF 在 0 轴上方 → 只做多；0 轴下方 → 只做空。是否明确标注？
- [ ] **刀口舔血检查**（§二十二 / §3.5）：15M 信号方向是否与 4H 方向一致？不一致且 4H 无衰竭 → 是否明确写"不做"？
- [ ] **中阴识别**（§十四 / §7.1）：若 4H 刚完成背驰/盘整背驰，是否标注"4H 处于中阴，方向待确认"？是否判断中阴健康/危险？

---

## 二、结构完整性

- [ ] **4H**：分型 → 笔 → 线段 → 中枢 → 走势类型 → 背驰 → MACD 方向 → 结论，完整推导
- [ ] **1H**：同上完整流程 + 明确中枢 ZD/ZG + 当前价格 vs 中枢位置 + 三买/三卖判断 + **奔走型反弹风险评估**（§三十三）
- [ ] **15M**：同上 + 买卖点类型 + 背驰比较对象 + 区间套 + 多义性全列

---

## 三、定义可审计

- [ ] **中枢**：有明确 ZD/ZG 价位（来自用户提供或估算 + 标注"未确认，误差约 ±10-20 点"）；哪三段次级别走势？ZD/ZG 怎么算的？
- [ ] **买卖点**：类型明确（第几类）、依据明确（哪个中枢、哪段离开、哪段回拉/回试）
- [ ] **背驰**：比较对象明确（哪两段、围绕哪个中枢、A/B 是否同级别、C 段是否创新高/低）；标注力度依据
- [ ] **线段终结**：用哪种情况判断的（无缺口/有缺口）？特征序列是什么？
- [ ] **止损逻辑**（§3.3）：止损触发原因是"结构破坏"，不是"亏了 X 点"。是否明确写出入场依据的结构及破坏条件？

---

## 四、诚实性

- [ ] 背驰判断标注了"视觉判断，待数值确认"还是"数值确认"
- [ ] 看不清的地方标注了"未确认"
- [ ] 多义性情况列出了多种解读（主解读 + 备选解读 + 各自成立前提 + 当前倾向）
- [ ] ZD/ZG 未提供时标注了"估算，误差约 ±10-20 点"

---

## 五、一致性

- [ ] 中枢位置前后一致（4H/1H/15M 各自独立，不互相矛盾）
- [ ] 买卖点判断前后一致
- [ ] 方向结论前后一致（4H → 1H → 15M 层层传递，不跳级）

---

## 六、风控与执行

- [ ] **入场价敏感性**（§3.2）：实际入场价偏差 ≥ 5 点，是否重新计算手数？
- [ ] **单笔风险上限**（§3.1）：净值 < $100k 时 ≤ 4%，≥ $100k 时 ≤ 2%
- [ ] **日/周纪律**（§3.4）：每天最多 1 笔；连续亏损 2 天 → 第 3 天强制观望
- [ ] **三级共振**：4H 方向明确 + 1H 中枢确认 + 15M 买卖点确认，全部满足才入场？不满足时是否明确写"今天不做"？
- [ ] **数据冲击规则**（§六）：重大数据后是否先判断破坏级别、等对应买卖点、不追杀？
- [ ] **宁卖错不买错**（§十）：出现顶分型/卖点信号时是否优先执行离场？

---

## 七、交易计划（共振满足时）

- [ ] 入场价 / 止损价 / 第一止盈价 + 预估盈亏比
- [ ] 首仓手数计算：最大亏损 ÷ 止损点数 ÷ $10
- [ ] 结构止盈触发条件
- [ ] 数据风控操作
- [ ] OpenClaw 监控价位（加仓关注价 / 止盈关注价）
- [ ] 分析结尾输出审计触发提示

## 标准追问模板

**追问中枢：**
> "你说的这个中枢，由哪三段次级别走势重叠构成？ZD/ZG 怎么算出来的？"

**追问买卖点：**
> "你说这里是 X 买/X 卖，请只证明这个买卖点，不要给交易建议。哪段是离开中枢？哪段是回拉？为什么算不回中枢？"

**追问背驰：**
> "你说出现背驰，比较的是哪两段？围绕哪个中枢？A/B 是否同级别？C 段是否创新高/低？是视觉判断还是数值确认？"

**追问刀口舔血：**
> "大级别走势方向是什么？大级别有无衰竭迹象（背驰/盘整背驰）？15M 信号方向与 4H 一致吗？"

**追问中阴：**
> "前一走势背驰点在哪？后续中枢在什么范围？健康还是危险？"

**追问线段：**
> "用哪种情况判断的（无缺口/有缺口）？特征序列元素是什么？第一二元素有无重叠？"

**追问止损：**
> "入场依据的结构是什么？什么条件下算结构破坏？止损价对应的是哪个结构位？"

**一致性检验（选做）：**
> 同一组图用不同问法再问一次（"只给结构，不给建议"），核心结论应保持一致。
