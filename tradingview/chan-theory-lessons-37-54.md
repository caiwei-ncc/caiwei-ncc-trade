## 第37课：背驰的再分辨

### ★★★ 核心前提：没有趋势，没有背驰

不是任何 a+A+b+B+c 形式的走势都有背驰。**只有当 a+A+b+B+c 构成趋势时，才能讨论背驰**。

**趋势的必要条件**：A、B 是同级别的中枢。若 A 的级别比 B 大，则不构成趋势，最多只能有盘整背驰。

### ★★★ 对 c 段的严格要求

| 条件 | 说明 |
|------|------|
| **c 必须是次级别** | c 至少包含对 B 的一个第三类买卖点 |
| **b 可以小于次级别** | 力度最大的 b 可以是连续缺口；b 在级别上不能大于 c |
| **c 必须创新高/新低** | 否则按盘整背驰处理 |

### ★★★ 区间套在 c 段中的应用

c 内部至少包含两个次级别中枢，构成次级别趋势时可继续套用 a+A+b+B+c 形式进行次级别趋势背驰分析，形成**区间套**。

### 实操要点（镑美适用）
- 在 4H 图上判断趋势背驰前，**必须先确认两个中枢 A、B 是同级别的**。
- 确认 c 段创了新高/新低后，再用 MACD 辅助比较。
- 如果 b 段力度很弱而 c 段力度很强，即使出现背驰信号也要**极度谨慎**。

## 第38课：走势类型连接的同级别分解

### ★★★ 同级别分解的定义与唯一性

**同级别分解**：把所有走势按一个固定级别的走势类型进行分解。具有**唯一性**。

### ★★★ 同级别分解的特殊规则

| 规则 | 同级别分解 | 非同级别分解 |
|------|-----------|-------------|
| 中枢延伸 | **不定义中枢延伸** | 允许中枢延伸 |
| 盘整+盘整连接 | **允许** | 不允许 |

### ★★★ 机械化操作程式

**向上段（先买后卖）**：下跌背驰买入 → 盘整背驰/背驰卖出 → 回调不破前低再买入 → 重复直到不创新高或盘整背驰

**向下段（先卖后买）**：所有操作反过来

**最大韵律**：向上段先买后卖，向下段先卖后买。韵律错了操作就一团糟。

### 实操要点（镑美适用）
- 按 1H 级别做同级别分解，只做上涨和盘整类型，避开下跌类型。
- 每段 1H 走势内部用 15M 级别判断背驰来精确定位买卖点。

## 第39课：同级别分解再研究

### ★★★ 核心操作规则

Ai 与 Ai+2 之间比较力度，用盘整背驰方法决定买卖点。

### ★★★ 两种基本图形分类
1. 同向段持续不背驰且不回破关键点 → 走势延续
2. Ai 与 Ai+2 之间盘整背驰 → 走势将转折

### 实操要点（镑美适用）
- 在 15M 图上标记各段走势，每次新段完成后与前一同向段比较力度。
- 出现盘整背驰 → 执行反向操作。

## 第40课：同级别分解的多重赋格

### ★★★ 小级别自动换档到大级别

当小级别走势发展到满足大级别分解条件时，可切换到大级别继续操作。

### 实操要点（镑美适用）
- 15M 操作遇到明显 1H 级别走好信号，可将部分仓位换档到 1H 级别。
- 但日内强制清仓约束下，只在有足够时间发展时才换档。

## 第41课：没有节奏，只有死

### ★★★ 市场节奏的唯一标准

**买点买、卖点卖**——市场唯一的节奏。贪婪与恐惧阻止你倾听。

### ★★★ 刀口舔血的定义

当小级别买卖点和大级别走势方向相反，而大级别走势没有任何衰竭迹象 → 参与小级别买卖点 = 冒大级别延续的风险。

### ★★★ 三次机会原则

市场给你三次改错机会：第一类 → 第二类 → 第三类买卖点。三次都错过 → 必须休息。

### ★★★ 节奏错了怎么办

**唯一正确做法：停止操作**，调节好心态再继续。

### 实操要点（镑美适用）
- **绝不追涨杀跌**。
- 如果 4H 方向看空而在 15M 上想做多 = **刀口舔血**（除非 4H 有衰竭迹象）。
- 连续两次亏损 = 节奏可能错了 → 第三天强制休息。

## 第43课：有关背驰的补习课

### ★★★ 两种不同的转折方式

**方式一：背驰级别 = 当下走势级别**
- 至少把走势拉向最后一个中枢，跌破/升破关键点是**必然的**

**方式二：背驰级别 < 当下走势级别**
- 不必然保证走势回到最后一个同级别中枢
- 需先形成更大中枢后才能进一步判断

### ★★★ 走势类型分解原则

> **一个某级别的走势类型中，不可能出现比该级别更大的中枢。**

### 实操要点（镑美适用）
- 在 1H 趋势中出现 15M 背驰时，不要急于判定为 1H 反转。
- **15M 背驰 ≠ 1H 反转**，但 15M 背驰 + 后续中枢出现第三类买卖点 → 可能性大增。

## 第44课：小级别背驰引发大级别转折

### ★★★ 缠中说禅小背驰-大转折定理

> **小级别顶背驰引发大级别向下的必要条件**：该级别走势的最后一个次级别中枢出现第三类卖点。
>
> **小级别底背驰引发大级别向上的必要条件**：该级别走势的最后一个次级别中枢出现第三类买点。

**注意**：这只是**必要条件，不是充分条件**。

### ★★★ 核心结论

小级别背驰后，如果能在最后一个次级别中枢正常震荡（未出现第三类卖点），就**不可能**转化成大级别转折。

### 实操要点（镑美适用）
- 15M 背驰后，观察最后一个 5M 中枢：
  - 5M 中枢出现第三类卖点 → 预警，准备减仓
  - 5M 中枢正常震荡 → 趋势大概率继续

## 第45课：持股与持币，两种最基本的操作

### ★★★ 操作的本质是等待

- 买点买入后 → 等待卖点
- 卖点卖出后 → 等待买点

### ★★★ 走势类型是生长出来的

不可能事先确认走势类型将以哪种方式完成。唯一需要做的：观察当下走势，让市场去选择。

### 实操要点（镑美适用）
- 入场后观察走势类型的生长状态，而非盯盈亏。
- "等待"本身就是操作的一部分。

## 第46课：每日走势的分类

### ★★★ 日内走势三大分类

**第一类：只有一个中枢（平衡市）**
- 弱平衡市：前三根 30 分钟 K 线出现当天高点
- 强平衡市：前三根 30 分钟 K 线出现当天低点
- 转折平衡市：前三根不出现当天高低点

**一般规律**：前三根 30 分钟 K 线决定全天波动区间。

**第二类：两个中枢（单边走势）**
- 变盘时间在**中午收盘前后 30 分钟内**

**第三类：没有中枢（最强单边走势）**
- 极为罕见，可能是骗线也可能是大级别强势趋势

### 实操要点（镑美适用）
- 伦敦开盘后前 90 分钟基本决定当天框架。
- 前 90 分钟出现高点 → 弱平衡市，做空更安全。
- 前 90 分钟出现低点 → 强平衡市，做多更安全。

## 第47课：一夜情行情分析（实战范本）

### ★★★ 当日走势分类实战应用流程
1. 用前三根 30 分钟 K 线判断日内分类
2. 中枢出现后缩小可能性
3. 用中枢震荡判断买卖点

### ★★★ MACD 级别选择原则

**先定好比较哪两段走势，然后再选择看几分钟的 MACD 更适宜。** 不是 30 分钟走势就一定看 30 分钟 MACD。

### 实操要点（镑美适用）
- 日内中枢确立后，用中枢震荡方法判断后续走势。
- MACD 辅助时选最清晰的级别去看。

## 第48课：暴跌，牛市行情的一夜情

### 核心观点
- 暴跌在趋势中就如同一夜情，猛烈但短暂。
- V 型顶在大型走势中基本不构成真正的顶部。
- 暴跌后出现的买点才是真正的机会，但要等结构确认。

### 实操要点（镑美适用）
- 突发暴跌时先判断级别（15M/1H/4H），不恐慌性平仓。
- 按预设止损执行即可。

## 第49课：利润率最大的操作模式

### ★★★ 进入市场的三大分类

找当下之前最后一个操作级别中枢：

| 情况 | 操作 |
|------|------|
| **当下在中枢之中** | 不操作，等演化 |
| **当下在中枢之下，未出现第三类卖点** | 中枢震荡 → 找盘整背驰买点 |
| **当下在中枢之下，已出现第三类卖点** | 用背驰确定买点或等新中枢 |
| **当下在中枢之上，未出现第三类买点** | 等待 |
| **当下在中枢之上，已出现第三类买点** | 刚形成时可介入 |

### ★★★ 缠中说禅第一利润最大定理

> 对于固定交易品种，在确定操作级别下，缠中说禅操作模式的利润率最大。

### ★★★ 中枢震荡差价的原则
- 中枢上方减仓，中枢下方加仓
- 中枢完成后的向上移动时不做差价，满仓
- 出现第三类卖点后不回补
- **绝不追涨杀跌**

### 实操要点（镑美适用）
- 每次打开图，**第一步找最后一个 1H 中枢**，判断价格相对位置。
- 中枢震荡的差价机会是日内交易最安全的操作模式。

## 第52课：炒股票就是真正的学佛

### ★★★ 绝对的预测

当一个 30 分钟顶背驰出现后，后面**必然**是一个 30 分钟级别下跌或扩展成 30 分钟级别以上的盘整。这是被理论绝对保证的。

### 实操要点（镑美适用）
- 1H 顶背驰出现 → **必然**至少一个 1H 盘整或下跌。
- 操作中战胜的不是市场，而是自己的贪嗔痴疑慢。

## 第53课：三类买卖点的再分辨

### ★★★ 三类买卖点各自的角色

| 类型 | 角色 |
|------|------|
| **第一类** | 该级别的背驰点，最大多数情况的最佳买卖点 |
| **第二类** | 补充第一类；小级别转大级别时是最佳买卖点 |
| **第三类** | 告诉你中枢何时结束 |

**第二、三买卖点之间 = 中枢震荡**，参与只能用低级别买卖点。

### ★★★ 第二类与第三类买点重合

若第二类买点恰好构成第三类买点 → 往往预示**大级别上涨**。

### 实操要点（镑美适用）
- 三类买卖点都不能偏废。
- 在 1H 中枢震荡期间，用 15M 买卖点做差价。
- 三级共振本质上就是在不同级别上同时满足买卖点条件。

## 第54课：一个具体走势的分析（实战分解范本）

### ★★★ 分解的基本原则
1. 以某级别分析时，次级别视为线段
2. 两个同级别中枢之间必须有次级别走势连接

### ★★★ 分解的多样性与选择原则
- 多种合规分解都不违反理论
- 选择对当下操作最有指导意义的分解

### ★★★ 中枢震荡的操作铁律
- 向上盘整背驰 → 抛
- 向下盘整背驰 → 回补
- **绝不追涨杀跌**

### 实操要点（镑美适用）
- 走势分解严格遵循原则，特别是"两个同级别中枢之间必须有次级别连接"。
- 多种分解时选对操作最有指导意义的。
- **绝对不在中枢震荡中追涨杀跌**。
- 大级别中枢突破时，小级别第三类买卖点就可介入，但要确认是真突破。
