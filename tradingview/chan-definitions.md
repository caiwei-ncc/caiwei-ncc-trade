# 缠论形式化定义手册

> **用途**：AI 分析时强制加载的可判定定义文件
> **来源**：以《教你炒股票》相关课程为主进行提炼；若定义存在课程口径差异或实务简化，分析时必须显式说明
> **原则**：每条定义给出"是什么"+"怎么判"+"常见错误"，可直接作为审计标准

## 一、包含关系（第62/65课）

### 定义
两根相邻K线，其中一根的高低点**完全在另一根范围内**。

判定条件（设左K线 [h1, l1]，右K线 [h2, l2]）：
- h1 ≥ h2 且 l1 ≤ l2 → 右被左包含
- h2 ≥ h1 且 l2 ≤ l1 → 左被右包含

### 处理规则

| 当前走势方向 | 合并方法 | 记忆口诀 |
|------------|---------|---------|
| **向上** | 新高 = max(h1, h2)，新低 = max(l1, l2) | 取高高 |
| **向下** | 新高 = min(h1, h2)，新低 = min(l1, l2) | 取低低 |

### 关键规则
- **不符合传递律**：A包含B、B包含C，不代表A包含C（第65课）
- **必须按顺序逐根处理**：合并后的新K线继续与下一根比较
- **方向判定**：看当前处理位置之前的走势方向（前一根K线相对再前一根的方向）

### 方向未明时的处理
- 若当前包含关系处的前置方向无法明确判定，必须标注"方向未明，包含关系处理存疑"
- 不得强行假设方向后继续高置信度分析

### 常见错误
- ❌ 跳过包含关系直接找分型
- ❌ 一次性扫描所有K线而非逐根处理
- ❌ 不区分方向就合并
- ❌ 方向不明时强行合并

## 二、分型（第62课）

### 定义（在处理完包含关系后的K线上判断）

**顶分型**：连续三根K线，第二根的**高点是三根中最高的**，且第二根的**低点也是三根中最高的**。

**底分型**：连续三根K线，第二根的**低点是三根中最低的**，且第二根的**高点也是三根中最低的**。

### 判定条件

顶分型（设三根K线分别为 K1, K2, K3）：
```
h2 > h1 且 h2 > h3
l2 > l1 且 l2 > l3
```

底分型：
```
l2 < l1 且 l2 < l3
h2 < h1 且 h2 < h3
```

### 常见错误
- ❌ 在未处理包含关系的原始K线上直接找分型
- ❌ 只看高点不看低点（如高点最高但低点不是最高，不构成顶分型）

## 三、笔（第62/65课）

### 定义
- **上升笔** = 底分型 → （至少一根K线）→ 顶分型
- **下降笔** = 顶分型 → （至少一根K线）→ 底分型

### 严格条件
1. 顶分型和底分型之间**至少有一根独立K线**（不算分型共用的K线）
2. 上升笔的顶分型高点**必须高于**底分型高点
3. 下降笔的底分型低点**必须低于**顶分型低点

### 特殊情况（第65课）
- 两个相邻顶分型之间无底分型 → 取**较高的那个**顶分型，忽略较低的
- 两个相邻底分型之间无顶分型 → 取**较低的那个**底分型，忽略较高的
- 处理后：先顶后底 = 下降笔；先底后顶 = 上升笔

### 常见错误
- ❌ 顶底之间没有独立K线就连笔
- ❌ 把相邻的两个同类分型都保留

## 四、线段（第62/65/67/71课）

### 4.1 线段成立条件
- 线段至少由**三笔**组成
- 前三笔必须有**重叠部分**（即上下上或下上下三笔的区间有交集）
- 不满足以上两个条件的笔组合**不构成线段**

### 4.2 线段终结确认条件（第67课特征序列法）

**特征序列**：
- 向上线段 → 特征序列 = 所有**向下笔**
- 向下线段 → 特征序列 = 所有**向上笔**

**标准特征序列**：对特征序列元素按包含关系处理后的序列

**线段结束的两种情况**：

| 情况 | 条件 | 结束位置 |
|------|------|---------|
| **第一种（无缺口）** | 标准特征序列的分型第一、二元素间**无缺口**（有重叠） | 分型的最高/最低点处 |
| **第二种（有缺口）** | 第一、二元素间**有缺口**（无重叠） | 需从分型最高点开始的反向序列出现反向分型才确认 |

### 关键规则
- **线段只能被线段破坏**，单纯一笔不能破坏线段（第64课）
- 线段划分具有**唯一性**（第67课）
- 精度一旦预设必须一路保持（第64课）

### 线段被破坏的数学条件（第65课）
- 向上线段：存在第 i 和第 j 个特征序列元素（j ≥ i+2），使 dj ≤ gi → 被破坏
- 向下线段：存在第 i 和第 j 个特征序列元素（j ≥ i+2），使 gj ≥ di → 被破坏

### 常见错误
- ❌ 只有两笔就当成线段
- ❌ 三笔无重叠也当成线段
- ❌ 用简单的"三笔有重叠"替代特征序列法判断线段终结

## 五、中枢（第17/18/20课）

### 定义
> 某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

### ZD/ZG 计算
设三个连续次级别走势的高低点分别为 [h1,l1], [h2,l2], [h3,l3]：
```
ZG = min(h1, h2, h3)  ← 中枢上沿（三段高点中最低的）
ZD = max(l1, l2, l3)  ← 中枢下沿（三段低点中最高的）
```

**中枢成立条件**：ZD < ZG（有重叠部分）

### 中枢的四个关键指标（第20课）

| 指标 | 定义 | 含义 |
|------|------|------|
| **ZG** | min(g1, g2, ...) | 中枢上沿 |
| **ZD** | max(d1, d2, ...) | 中枢下沿 |
| **GG** | max(g1, g2, ...) | 中枢波动区间最高点 |
| **DD** | min(d1, d2, ...) | 中枢波动区间最低点 |

### 中枢的三种后续演变

| 演变 | 条件 | 结果 |
|------|------|------|
| **延伸** | 新的次级别波动仍触及 [ZD, ZG] | 中枢不变，围绕震荡 |
| **新中枢** | 离开原中枢后，形成与原中枢**不重叠**的同级别中枢 | 构成趋势中的新中枢；若波动区间与原中枢重叠则按级别扩展处理，不得直接判为趋势 |
| **级别扩展** | 新中枢的波动触及前中枢波动区间 | 产生更大级别中枢 |

### 中枢破坏/突破（第18课定理三）
> 一个次级别走势离开中枢后，其后的次级别回抽走势**不重新回到中枢内** → 中枢被突破

### 常见错误
- ❌ 用两段走势就定中枢（至少需要三段）
- ❌ ZG/ZD 算错方向（ZG 取 min 不是 max）
- ❌ 把中枢波动范围 [DD, GG] 当作中枢区间 [ZD, ZG]

## 六、走势类型（第17课）

### 精确定义

| 类型 | 定义 |
|------|------|
| **盘整** | 完成的走势类型只包含**一个**中枢 |
| **趋势（上涨）** | 完成的走势类型包含**两个以上**依次向上的中枢（中枢间不重叠，后中枢 DD > 前中枢 GG） |
| **趋势（下跌）** | 完成的走势类型包含**两个以上**依次向下的中枢（中枢间不重叠，后中枢 GG < 前中枢 DD） |

### 走势分解定理
- **定理一**：任何走势可分解为盘整、下跌、上涨三种类型的连接
- **定理二**：任何走势类型至少由三段以上次级别走势类型构成

### 走势终完美（第17/69课）
> 任何级别的任何走势类型**终要完成**。这是100%的理论保证。

### 常见错误
- ❌ 只有一个中枢就说是"趋势"
- ❌ 两个中枢有重叠也说是"趋势"（应为级别扩展）
- ❌ 盘整中讨论趋势背驰（盘整只有盘整背驰）

## 七、三类买卖点（第12/17/20/21/53课）

### 第一类买点
**定义**：下跌趋势中，最后一段下跌出现**趋势背驰**的位置。
- 必须对应 a+A+b+B+c 结构，且 c 段创新低、力度弱于 b 段
- 必须在趋势中（至少两个同向中枢后）
- MACD 辅助：C 段柱子面积小于 A 段

### 第一类卖点
**定义**：上涨趋势中，最后一段上涨出现**趋势背驰**的位置。

### 第二类买点
**定义**：第一类买点出现后，第一次反向走势的结束位置。
- **前提**：第一类买点已成立；其后第一次次级别回拉结束，且不破坏第一类买点的转折结构（不创新低）
- 缠中说禅买点定律：**大级别的第二类买点由次一级别相应走势的第一类买点构成**（第14/17课）
- 比第一类更安全

### 第二类卖点
**定义**：第一类卖点出现后，第一次反弹走势的结束位置。

### 第三类买点（第20课 ★★★）
**严格定义**：
1. 存在一个中枢 [ZD, ZG]
2. 一个次级别走势类型**向上离开**中枢（价格超过 ZG）
3. 随后一个次级别走势类型**回试**
4. 回试的**最低点不跌破 ZG**
5. **必须是第一次**回试不回中枢

### 第三类卖点（第20课 ★★★）
**严格定义**：
1. 存在一个中枢 [ZD, ZG]
2. 一个次级别走势类型**向下离开**中枢（价格低于 ZD）
3. 随后一个次级别走势类型**回抽**
4. 回抽的**最高点不升破 ZD**
5. **必须是第一次**回抽不回中枢

### 买卖点完备性定理（第21课）
> 市场必然产生赢利的买卖点，只有第一、二、三类。

### 三类买点关系
| 关系 | 结论 |
|------|------|
| 一买 vs 二买 | **不可能重合** |
| 一买 vs 三买 | **不可能重合** |
| 二买 vs 三买 | **可能重合**（出现时信号极强） |

### 常见错误
- ❌ 在盘整中找"第一类买点"（盘整只有盘整背驰，不是趋势背驰）
- ❌ 第三类买点不检查"是否第一次回试不回中枢"
- ❌ 混淆中枢区间 [ZD,ZG] 和波动区间 [DD,GG]，用错了判断标准

## 八、背驰（第15/24/27/37课）

### 8.1 结构前提（背驰的本体定义）

> **没有趋势，没有背驰**（第15课）。盘整中只有"盘整背驰"，不能叫"背驰"。

**趋势背驰的结构前提**：
- 结构：a + A + b + B + c，其中 A、B 是**同级别**中枢
- A 和 B 必须是同级别中枢（第37课）
- c 段必须是次级别走势（至少包含对 B 的一个第三类买卖点）
- c 必须创新高（上涨趋势）或新低（下跌趋势）
- c 段力度弱于 b 段 → **趋势背驰成立**

**盘整背驰的结构前提**：
- 围绕**一个**中枢的前后两段走势
- 后段力度弱于前段 → **盘整背驰成立**

**盘整背驰后的两种情况**：
| 情况 | 条件 | 操作 |
|------|------|------|
| 回落不回中枢 | 回试最低点不跌破 ZG | 构成第三类买点 → 买回来 |
| 回落回到中枢 | 重新跌回 [ZD, ZG] 内 | 继续盘整 → 等待 |

**背驰后的三种可能（第29课）**：
| 情况 | 描述 | 力度 |
|------|------|------|
| 反弹不回最后中枢 | 中枢级别扩展 | 最弱 |
| 反弹回到中枢内 | 更大级别盘整 | 中等 |
| 反弹超越中枢并形成第三类买点 | 反趋势 | 最强 |

### 8.2 MACD 辅助确认方法（工具，不是定义本身）

> MACD 是辅助确认背驰的工具，不是背驰的定义。结构前提不满足时，MACD 再弱也不构成背驰。

**标准方法（第24课）**：
1. 识别 **A 段**（第一段同向走势）
2. 识别 **B 段**（中间中枢，MACD 黄白线回拉 0 轴附近）
3. 识别 **C 段**（第二段同向走势）
4. 比较 C 段与 A 段的 **MACD 柱子面积**
5. C < A → **辅助确认背驰**

**力度比较的级别区分（第56课）**：
| 级别 | 比较方法 |
|------|---------|
| 线段以下级别 | 只比柱子面积 |
| 1分钟及以上级别 | 同时看黄白线回抽 0 轴情况 |

**截图 vs 数值场景**：
| 场景 | 允许的判断 |
|------|-----------|
| 仅有截图 | 只能做视觉力度对比，标注"视觉判断，待数值确认" |
| 有 MACD 具体数值 | 可做精确数值比较，写明各段具体数值 |

### 常见错误
- ❌ 盘整中说"趋势背驰"
- ❌ A 和 B 不是同级别就做背驰比较
- ❌ c 段没创新高/新低就判定为趋势背驰（应按盘整背驰处理）
- ❌ 比较的两段不围绕同一中枢
- ❌ 结构前提不满足时仅凭 MACD 柱子缩小就判定为背驰

## 九、区间套（第27/37/61课）

### 定义
> 通过不同级别背驰段的逐级收缩范围，精确确定大级别转折点。

### 标准流程
1. **大级别**：找到背驰段（趋势背驰或盘整背驰）
2. **次级别**：在该背驰段内部找到中枢，找前后段的背驰
3. **更次级别**：继续向内套用
4. **最低操作级别**：在最终背驰段范围内确认转折点

### 我们的三级框架对应
```
4H 找背驰段 → 1H 在该段内找背驰 → 15M 精确定位入场点
```

### 常见错误
- ❌ 跳级套（从 4H 直接到 15M，跳过 1H）
- ❌ 外层不存在背驰就开始向内套

## 十、多义性（第33课）

### 三种多义性
1. **中枢延伸造成的**：中枢延伸不超过 5 段次级别走势；6 段以上构成更大级别中枢
2. **模本简略造成的**：不同级别图是不同精度的显微镜
3. **走势分析中的多种合理释义**（有实质意义）

### 处理原则与输出要求
- 同一价格位置在不同级别中枢下有不同解读时，**全部列出**
- 输出必须包含：
  1. **主解读**：最倾向的判断
  2. **备选解读**：其他合理的判断
  3. **各自成立的前提条件**
  4. **当前更倾向哪一种及原因**
- 不强行给单一结论

## 十一、小背驰-大转折定理（第44课）

### 定理
> 小级别顶背驰引发大级别向下的**必要条件**：该级别走势的最后一个次级别中枢出现第三类卖点。
> 小级别底背驰引发大级别向上的**必要条件**：该级别走势的最后一个次级别中枢出现第三类买点。

### 关键
- 这只是**必要条件，不是充分条件**
- 如果最后一个次级别中枢正常震荡（未出现第三类买卖点），就**不可能**转化成大级别转折

### 我们的应用
- 15M 背驰后，观察最后一个 5M 中枢：
  - 出现第三类卖点 → 预警，可能大级别转折
  - 正常震荡 → 趋势大概率继续
