## 第91课：走势结构的两重表里关系1

### 核心观点

- 判断走势如同中医看病，分**未病—欲病—已病**三阶段，对应相应级别的**第一、二、三类买卖点**。
- 级别类比疾病级别：1分钟=小感冒，5分钟=小流行，30分钟/日线=中级调整，周/月线=重症，季/年线=植物人级别。
- 走势分解的配件分两种：**一、能构成中枢的**（线段、各级别走势类型）；**二、不能构成中枢的**（仅有笔）。笔是唯一不能构成中枢的配件，这是笔与线段及更高级别走势类型的根本区别。
- **笔在不同时间周期K线图上的相应判断，构成第一重表里关系。**
- 可用两个变量构成的数组精确定义任何当下走势在任何时间周期里的状态，可用**4的N次方**个状态矩阵描述所有可能走势。
- 只关心**三个连续级别**（如1、5、30分钟）即对应**64种状态**，与易经六十四卦对应。
- 光有笔这重表里关系不足以精确诊断，还需研究**另外的表里关系**以及不同表里关系之间的**生克关系**。

### 关键定义

**缠中说禅笔定理**：任何的当下，在任何时间周期的K线图中，走势必然落在一确定的具有明确方向的笔当中（向上笔或向下笔），而在笔当中的位置，必然只有两种情况：一、在分型构造中；二、分型构造确认后延伸为笔的过程中。

**笔状态数组**：用(方向, 阶段)两个变量表示。
- 方向：1=向上笔，-1=向下笔
- 阶段：0=分型构造中，1=分型确认延伸为笔的过程中

**四种笔状态**：
- **(1, 1)**：向上笔在延伸中
- **(-1, 1)**：向下笔在延伸中
- **(1, 0)**：向上笔出现顶分型构造
- **(-1, 0)**：向下笔出现底分型构造

**状态连接规则**（不可随意连接）：
- (1, 1) → 只能连接 (1, 0)
- (-1, 1) → 只能连接 (-1, 0)
- (1, 0) → 可连接 (1, 1) 或 (-1, 1)
- (-1, 0) → 可连接 (-1, 1) 或 (1, 1)

**走势病情矩阵**：按1分钟、5分钟、30分钟、日、周、月、季、年共8个级别分类，矩阵有8行，每行为对应级别的状态数组，共4^8种可能状态。

### 实操要点（镑美适用）

1. **多周期笔状态联合判断**：同时标注GBP/USD的M1、M5、M30三个周期的笔状态数组，形成3行状态矩阵（64种状态之一），作为当下走势的精确定位。
2. **高级别过滤低级别噪音**：若M5处于(1,1)或(-1,1)状态，M1的波动无须关心——它不足以改变M5的笔状态。日内操作至少关心M5图，可过滤M1的无效波动。
3. **未病—欲病—已病的实战判断**：
   - 对M5的(1,1)状态，M1出现(1,0)是**小警告**（未病）；
   - M1的(1,0)被确认为(-1,1)，且导致M5出现(1,0)在形成中，则**欲病向已病发展**；
   - M5的(1,0)确认向(-1,1)发展时，**确认已病**（此时M5级别的笔方向已转变）。
4. **恶劣程度排序**（以下跌为例，做多者回避）：
   - 最恶劣：高级别(-1,1) + 低级别(-1,1) → **绝对回避**
   - 次恶劣：高级别(-1,1) + 低级别(-1,0) → 技术高者可操作
   - 第三恶劣：高级别(-1,0) + 低级别(-1,1) → 技术高者可操作
   - 第四恶劣（转机可能）：高级别(-1,0) + 低级别(-1,0) → 可能出现转机
5. **镑美应用**：每日开盘前，标注H1与M15（或M30与M5）的笔状态数组，判断当前属于64种状态中的哪一种，据此决定是否参与及参与方向。

## 第92课：中枢震荡的监视器

### 核心观点

- 中枢震荡最终一定以某级别的**第三类买卖点**结束。
- 需要一个监控工具判断震荡是**逐步走强还是逐步走弱**，并预估每次震荡高低点的大致区间。
- **Zn最终肯定要超越A或B**（中枢区间上下沿），否则永远不会出现第三类买卖点——这是必然的。
- Zn超越A或B并不一定意味着出现第三类买卖点，可能多次超越，只有最后一次才构成第三类买卖点。但一旦出现超越，就是**变盘的重大提醒**。
- 如果超越没构成第三类买卖点，一般都将构成**中枢震荡级别的扩展**（概率极高，非100%）。
- **Zn缓慢提高但无力突破B**的走势，小心**上升楔形诱多**；反之为下降楔形诱空。
- 中枢震荡中次级别如果是**趋势类型**，且Zn出现相应配合，特别是**最后一个次级别中枢在原中枢之外**时，一旦下一个次级别走势在该次级别中枢区间完成，**变盘即将发生**。
- 结合**布林通道**的时间把握，对变盘的预见性极高。

### 关键定义

**震荡中轴Z**：中枢区间的一半位置（中枢区间[A, B]的中点）。

**次级震荡中轴Zn**：每一个次级别震荡区间的一半位置，依次标注。最标准状态下Zn=Z，但这是特例。

**Zn与Z的关系**：
- Zn在Z之上 → 震荡偏强
- Zn在Z之下 → 震荡偏弱
- A、Z、B三条线等距，Zn的波动连成曲线构成监视中枢震荡的技术指标。

**Zn数量限制**：不会超过9个数据，超过则次级别要升级。

### 实操要点（镑美适用）

1. **构建Zn指标**：在GBP/USD的M30或H1中枢震荡中，标记中枢区间[A, B]及中轴Z，然后逐个标注每次次级别（M5或M15）震荡的中点Zn，观察Zn的移动方向。
2. **强弱判断**：Zn序列持续在Z之上→中枢震荡偏多头，可偏多操作；反之偏空。
3. **变盘预警**：
   - Zn首次超越B（上沿）→ 向上变盘预警，警惕第三类买点形成；
   - Zn首次超越A（下沿）→ 向下变盘预警，警惕第三类卖点形成。
4. **诱多/诱空识别**：Zn缓慢走高但始终无力突破B→上升楔形诱多，不追高；Zn缓慢走低但始终不破A→下降楔形诱空，不追空。
5. **买入风险控制**：Zn在Z之下甚至在A之下时介入做多，风险很大——手脚不够麻利可能被堵死在交易通道中，无法顺利完成震荡操作。
6. **预估下一个震荡区间**：Zn变动通常相对平滑，可大致预计下一个Zn的区间，从而预估下一个震荡的高低点。

## 第93课：走势结构的两重表里关系2

### 核心观点

- 所有问题集中在**(1, 0)或(-1, 0)之后怎么办**——这两种状态之后都有(1, 1)和(-1, 1)两种可能。
- (1, 0)或(-1, 0)这个信号是**绝对明确、毫不含糊的**，任何人都可以唯一确定。同一信号对不同水平的人，处理方法不同。
- **三类操作者的应对策略**（以(1, 0)为例，(-1, 0)反之）：
  - **第一类：保守型**（震荡水平一般，胆子小，无时间/跑道）：足够周期的(1,1)后出现(1,0)→**先兑现成本，留下利润**让市场自行选择。剩余仓位：若出现(-1,1)，在向下笔结束后的向上笔不创新高时→清仓。更简单：看5周期均线，有效跌破就走。
  - **第二类：技术型**（震荡水平较好）：利用(1,0)后必然出现的震荡进行**先卖后买的短差操作**。若发现市场选择(-1,1)→最后一次不回补，完全退出。
  - **第三类：趋势延续判断**：若市场最终选择(1,1)→(1,0)区间极为重要，其上下两段(1,1)进行**力度比较**，后段力度小于前段=**明确见顶信号**，然后用区间套定位精确高点。
- **(1,0)出现的两种情形**：
  - 对应上涨出现明确背驰完全确认结束→震荡区间以上涨**最后一个中枢**为依据，围绕该区间=强震荡，离开该区间=弱震荡。弱震荡一旦确认，最好不参与，等(-1,0)再说。
  - 背驰未完全确认→继续观察。

### 实操要点（镑美适用）

1. 在M30图上标注笔状态，当出现(1,0)时，根据自身操作能力选择上述三类策略之一。
2. 日内交易者建议用第二类策略：(1,0)后做短差，先卖后买，发现(-1,1)则不回补。
3. (1,0)区间前后两段向上笔的力度比较（MACD面积或K线斜率），是判断趋势是否延续的关键。
4. (1,0)对应的走势类型级别需要看图确认：如H1上的(1,1)到(1,0)，必然对应一个小级别上涨，级别是M1还是M5还是M15需看具体图形。
5. **弱震荡规避**：若(1,0)后震荡区间离开上涨最后一个中枢范围→确认弱震荡→不参与，等(-1,0)即下跌笔的底分型构造再考虑。

## 第94课：当机立断

### 核心观点

- 学习缠论，第一层次就是达到**当机立断**。
- 机会可以预先分析，但不是预测，而是建立在**完全分类基础上的边界分划**，这分划来自理论的纯数学构造，**唯一性与精确性保证了分类与边界的当下确认性**。
- 对任何走势，都可以根据理论**马上严格给出必然出现的机会**。市场机会与理论输出**严格一一对应**。
- 操作流程：罗列必然出现的机会→选择想参与的机会→准备通道/资金/一切安排→**设置退出的边界条件**→等待机会显现→当机立断。
- **级别**是脑中必须时刻有的概念。搞不清级别，一切操作无从谈起。
- 不会卖出就等于失去下次买入的机会。节奏之所以难，是**贪嗔痴疑慢**作怪。
- 对初学者必须**机械束缚**：5周均线、5日均线，分型后有效破了一定走。熟练者则由严格走势分类自动给出一切。

### 实操要点（镑美适用）

1. **机会罗列法**：每日开盘前，根据GBP/USD当前走势，用缠论严格列出后续必然出现的所有机会（如：M5线段背驰→M15中枢形成→M30第三类买卖点等），形成一个机会清单。
2. **选择与放弃**：根据自身当下状态（资金量、时间、心态），选择想参与的机会级别，主动放弃不适合的。
3. **退出条件预设**：参与任何机会前，必须先设好退出边界。例如做多M5背驰买点，退出条件=最后一个类中枢被跌破，或线段向上走势中出现类背驰/类盘整背驰。
4. **T+0优势**：外汇无T+1限制，比A股更适合当机立断的操作模式，背驰进场后可随时根据走势变化退出。
5. **初学者机械规则**：在M30图上，以5周期均线为参考，分型确认后有效破5均线一定平仓。这是最基本的纪律约束。

## 第95课：修炼自己

### 核心观点

- 要战胜市场，首先要了解市场的众生——市场是**合力**的，合力的是活生生的人。
- **市场就是要0投入去赚钱**。用第一笔利润作为资本，把本拿走，然后把利润变成巨大数字。
- 只要有**稳定的技术和操作**，初始投入多少根本不重要。1万元，10次翻倍就是1000万。
- 市场上的真正成功**以十年为单位**。
- 绝大多数人因**贪婪**而不断投入，因**恐惧**而落荒而逃，最终在高潮中被忽悠回来被煮。
- 市场中唯一需要考虑的就是**对操作水平的提高**，别人最多是你的陪练。
- 学习理论必须**彻底穷源**，然后在实践中不断升级。

### 实操要点（镑美适用）

1. **资金管理原则**：用一笔完全不影响生活的资金开始镑美交易，视为唯一资本，不追加。盈利后把本金撤出，只用利润操作。
2. **以十年为尺度**：不追求单笔暴利，追求操作水平的稳定提升。每次犯错后彻底复盘，找到根源。
3. **心态检查**：每次操作前问自己——是因为贪婪在追？还是因为恐惧在逃？如果是，就不操作。
4. **操作是唯一的评价标准**：不评论别人的交易，只专注提升自己的操作记录。

## 第96课：无处不在的赌徒心理

### 核心观点

- 赌徒心理最大特点：**预设一个虚拟目标，一个想象中的目标，完全无视市场本身**。
- 另一个特点：**怕失去机会，怕失去赚大钱的机会**——"万一走错了怎么办？万一还涨不就亏了？"诸如此类。
- 赌徒心理的常见表现：
  - 砍了又追，追了又砍，被无明业力牵引→被杀（杀红了眼）；
  - 不敢操作，看到机会害怕，起来后悔再追→被杀（5元不敢买，50元都敢买）；
  - 听消息，找捷径，以为有馅饼一定能拍着自己；
  - 带着"赚钱买房买车"的目标入市→市场不是慈善场所，是杀人的地方。
- 市场生存靠的是**平凡但能长久的赢利模式**，在一定的韵律中生长出利润。
- **错过了就错过了**，后面有无数机会。不需要把自己设计为超人。
- 如果两次尝试都输光，考虑退出市场N年后再来；如果第三次还不行，**和市场永远再见**。

### 实操要点（镑美适用）

1. **禁止虚拟目标**：不设"今天要赚X点"的目标，只按走势分类给出的买卖点操作。
2. **禁止追单**：错过的机会就放弃，等下一个理论输出的机会。5元不敢买的（GBP/USD没敢在背驰点入场），过段时间50元（等涨了100点）敢买→典型赌徒行为。
3. **禁止砍了又追**：每次操作必须有完整的分类依据和退出条件，不因情绪反复。
4. **建立韵律**：找到适合自己的操作级别（如M15或M30），按该级别的买卖点节奏操作，不越级、不频繁切换。
5. **输光机制**：给自己设定两次机会（两次本金），如果两次都亏完，停下来学习N个月再回来。

## 第97课：中医、兵法、诗歌、操作1

### 核心观点

- 市场操作最相似于**中医、兵法、诗歌**。
- 三步修炼：
  - **第一步**：彻底吃透理论——所有市场机会都会被理论一一对应地输出（如同中医学好理论体系）。
  - **第二步**：看机会的当下显发，对之**望闻问切**——如同医生不预测下一个病人的病症，只在病人到来时诊断。**市场机会到来时不需要预测点位、时间，只需要当下望闻问切。**
  - **第三步**：操作——如同中医开药方，是最难的9/10。因人、因时、因地而不同，没有两场一样的仗。
- **一定要看走势图**，那是世界上最昂贵的艺术品。站在热爱的角度培养鉴赏力，而非被赢亏之念蒙蔽眼睛。
- 所有不同中有一个共同的**"不患"基础**——市场运行的不患基础被缠论完全描述。在不患背景之上，每个人按自己能力开出不同药方（操作记录）。
- 在一般程度上必须**遵守买卖点原则**去买卖。但高水平者可以在买卖点偏移处买卖，以获得更多数量或达到更大目的——如同**格律诗的拗体**，先严格遵守格律，再自设声律。

### 实操要点（镑美适用）

1. **不预测，只诊断**：不预测镑美明天涨还是跌、涨到哪里，只在机会到来时用缠论工具当下判断。
2. **培养走势鉴赏力**：每天花时间纯粹看GBP/USD的走势图（多周期），像欣赏艺术品一样，培养对走势结构的直觉。
3. **初学阶段严格遵守买卖点**：不在非买卖点位置操作，不找借口"拗体"操作。只有在买卖点操作已完全得心应手后，才考虑偏移操作。
4. **操作是一生的修炼**：不追求速成，每次操作都是练习。错几次很正常，关键是在错误中提升鉴赏力。

## 第98课：中医、兵法、诗歌、操作2

### 核心观点

- **正确的思维方式**：背驰后，理论只能保证回拉原来的中枢。回拉之后如何→**完全分类**，根据每种分类对应的后果决定对策。不预测，只分类。
- 所有分析都在**严格的逻辑关系中**，是理论的必要结果。
- 如同打仗：不能**假设**明天某山头在自己手里，山头是否在手里是当下的。可以反复争夺，但不能假设未来某时刻100%在手里。
- 如同中医：不能认为药一定有效，效果是吃药后再望闻问切测出来的，药方要根据不断的望闻问切**增减调整**。
- **换言之，操作时，后续所有可能面对的情况与对策都必须了然，否则没资格操作。对于真正的操作者，没有任何情况是意外的——所有情况都被完全分类了，所有对策都事先有了，只等市场自己触及预设的开关。**

### 实操要点（镑美适用）

1. **完全分类思维训练**：每次进场后，立即列出后续所有可能走势及对应对策。例如：M5背驰买入后→分类1：继续向中枢上沿突进（持有）；分类2：回探破坏当前线段（止损或减仓）。
2. **不假设，只确认**：不假设镑美"应该"怎么走，只看当下走势是否触及分类边界。
3. **先于大盘选品种**（外汇应用：选择比大方向更强势的入场时机）：在背驰点进场时，选择先于主要趋势转折的时点，这样即使大方向回转，也有足够空间应对。
4. **消息无关性**：操作逻辑与是否有消息/数据无关。有利好竟然还走出破坏性走势→问题严重。操作只根据图形是否触及分类界限进行。

## 第99课：走势结构的两重表里关系3

### 核心观点

- **中阴状态**的存在反映了走势生长阶段的未确定性，是走势的**量子化特性**，不是理论不完善。
- 中阴状态不影响操作——都可看成**中枢震荡的整理**，按中枢震荡操作即可。
- 根据**结合律**，连接中枢的走势并不一定是完全的趋势类型，可以通过结合律调整划分，使连接中枢的走势保持最完美形态。
- **核心必然结论**：连接中枢的走势级别一定小于中枢。即**一个走势级别完成后，必然面临至少大一级别的中枢震荡**。例如5分钟上涨结束后，必然至少有一个30分钟中枢震荡。没有任何走势可以逃脱。
- 后续更大级别中枢震荡**必然至少落在前一走势类型的最后一个中枢范围里**——这是正常行为，此时中阴状态是健康的。
- 一旦中枢震荡**回到原走势类型的第二甚至更后中枢里**，中阴状态就是不健康、危险的。原走势最后一个中枢成为**关键指标位置**。

### 关键定义

**中阴状态**：走势生长阶段的未确定性状态。当一个走势类型结束后，首先形成一个中枢，进入中阴状态——此时不能对走势给出明确的级别划分。

**中阴健康判断标准**：
- 健康：后续中枢震荡落在前一走势的**最后一个中枢范围内** → 正常
- 不健康/危险：中枢震荡回到前一走势的**第二或更早中枢范围内** → 危险

### 实操要点（镑美适用）

1. **中阴状态的操作**：镑美在一段M30上涨结束后进入中阴状态时，按中枢震荡操作——高抛低吸，不急于判断级别。
2. **监控中枢震荡位置**：中阴期间形成的中枢，如果始终在前一上涨的最后一个中枢区间内→健康，可继续按原方向偏多操作。如果跌回到倒数第二个中枢区间→危险信号，应减仓或反向。
3. **级别完成后的必然预期**：一个M5上涨结束后，必然至少面临M15级别的中枢震荡。操作者必须为此预留应对空间，不可假设上涨会不经调整地延续。
4. **分型与中阴的结合**：H1分型出现时，对应的小级别走势中出现的中枢位置极为关键——几乎决定该分型是否是真正的顶或底。

## 第100课：中医、兵法、诗歌、操作3

### 核心观点

- **完全分类不是单层次的，必须是多层次的**。缠论的级别自然给出分类的层次，综合起来就是**立体的完全分类系统**。

- 如果你没法应付某个可能出现的分类（如第三类卖点），就**别参与**这个机会。

### 实操要点（镑美适用）

1. **三层次系统构建**：选择M30/M5/M1（或H1/M15/M5）三个层次，每个时间点标注三个层次各自的走势状态（趋势/盘整、中枢位置、背驰状态等）。

4. **实战推演示例**：镑美M30处于下跌中枢震荡，M5处于离开中枢向下移动中，M1出现一段下跌。→ 第一个必然且最先出现的变化=M1底背驰 → M1底背驰后回拉 → 分类1：构成M5第三类卖点（确认M5下跌）→ 分类2：不构成第三类卖点（M5中枢继续震荡）→ 据此决定参与深度。

## 第101课：答疑1

### 核心观点

#### 一、第二类买卖点

- **第二类买点定义**：第一类买点（背驰点）的次级别回抽结束后，再次探底或回试的那个次级别走势的结束点。
- 例如：5分钟底背驰后，第一类买点上去的1分钟走势结束后，回头的1分钟向下走势的结束点=第二类买点。

**第二类买点的三种情况**：
- **最强**：第二类买点刚好构成原下跌最后一个中枢的第三类买点（第二、三类买点合一）→对应V型反转，最有力度。
- **最弱**：第二类买点跌破第一类买点（比第一类买点更低）→一般构成盘整背驰，后面对应顺势平台到扩张平台等走势。
- **一般性**：第一、二、三类买点依次向上，一个比一个高。

- 只有在**回升的中阴状态下才有第一、二类买点**。中阴状态结束后，所有中枢震荡只存在**第三类买卖点以及中枢震荡买卖点**。

#### 二、走势必完美

- **走势必完美**=缠论给出的分型、笔、线段、不同级别走势类型所对应的递归函数，能将行情的任何走势**唯一地分解**。
- **唯一分解定理**在现代数学任何分枝中都是核心问题。
- 分型等于递归函数的a0，可以随意设计而不影响唯一分解定理的证明。但当前设计是所有可能设计中最好的——使笔出现的可能性最大并消除最多偶然因素。

### 实操要点（镑美适用）

1. **识别第二类买点的三种形态**：
   - 镑美M30底背驰（第一类买点）后上涨一段再回调→回调结束点是否高于第一类买点？如果刚好构成原中枢的第三类买点→最强，重仓做多。如果跌破第一类买点→最弱，轻仓或不参与。
2. **中阴状态结束的判断**：当中阴状态结束（确认了新的走势级别后），不再寻找第一、二类买点，只按第三类买卖点和中枢震荡买卖点操作。
3. **唯一分解的实战意义**：对任何一段镑美走势，都应该能给出唯一的缠论分解（笔→线段→走势类型），如果你的分解不唯一（两种分法都说得通），说明你对分型/笔/线段的定义理解有误。

## 第102课：再说走势必完美

### 核心观点

- 走势必完美对应的是一种**最特殊、最强有力的唯一分解**——类似自然数的**记数法分解**（如2进位、10进位）。
- 缠论的递归函数完美给出市场走势一个**类似记数法一样的唯一分解**，揭示了看似毫无规律的市场走势有着和自然数类似的整体结构。
- **级别极端关键**：递归函数是有级别的，级别依次升大。搞不明白级别=不明白缠论。
- **区间套方法**是走势必完美的重要应用——如果没有这个整体结构，区间套不会存在。区间套使买卖点的**精确定位**成为可能。
- 从1分钟到年共8个级别（加上线段与笔可更精细），但一般操作只需**1、5、30分钟三个级别**的分解，大资金可加上日级别。
- **任何走势都可唯一表示为 a1A1 + a5A5 + a30A30 的形式**。
- **核心结论**：任何高级别的改变必须先从低级别开始。绝不可能出现5分钟从下跌转折为上涨而1分钟还在下跌段中的情况。
- 由于走势必完美，走势**可以完全分类**，所有分类都有明确界限，任何走势都成为**可控的**——当下直接显现，不需要预测。
- **完全分类是级别性的，有明确点位界限，不是粗糙的上下平概念。理论完全数量化、精确化，不存在含糊之处。**

### 实操要点（镑美适用）

1. **三级别唯一分解**：将GBP/USD当下走势唯一表示为M1状态 + M5状态 + M30状态的组合，每个级别的状态是确定的（上涨/下跌/盘整，处于哪个中枢，是否背驰等）。
2. **区间套定位精确买卖点**：M30出现背驰信号→M5找对应的背驰点→M1找更精确的转折点。逐级缩小区间，精确到几个点的误差。这是缠论最强大的实战工具之一。
3. **级别传递规则**：做多时，必须看到M1先出现底背驰转折→M5随后出现转折信号→M30才可能改变。如果M1还在下跌延伸中，不可能M5已经转折——违反此规则的判断一定是错误的。
4. **数量化操作**：所有判断都有精确的点位界限（某个分型的高低点、某个中枢的区间），不存在"大概"、"差不多"的判断。

## 第103课：学屠龙术前先学好防狼术

### 核心观点

- 在没彻底掌握防狼术之前，别研究中枢、级别。防狼术至少保证不被大盘严重侵犯。

- 技术高的可以在**背驰情况下介入**（0轴以下的底背驰），但这是更高要求。

### 关键定义

**MACD防狼术**：选择一个你能力所决定的最低时间周期（如H1或M30），该周期MACD黄白线（DIF和DEA）在0轴以下=空头主导→不参与做多；在0轴以上=多头主导→不参与做空。

### 实操要点（镑美适用）

2. **执行规则**：
   - M30 MACD黄白线在0轴下方→不做多镑美，只观望或做空；
   - M30 MACD黄白线在0轴上方→不做空镑美，只观望或做多。
3. **0轴反抽确认**：黄白线跌破0轴后的第一次反抽0轴确认失败→确认空头主导，此后产生的所有杀伤力都是你应该回避的。
4. **这是最低要求**：连这都做不到就不要做交易。高手可以在0轴下方寻找底背驰机会，但初学者绝对不允许。

## 第106课：均线、轮动与缠中说禅板块强弱指标

### 核心观点

- 任何级别都有**最少延伸时间**：一笔至少延伸6个基本K线单位（因为需要顶底分型），据此可推算最少挑战的均线。
- 走势级别与均线虽无必然关系，但有**大致区间对应**。
- 经验法则：**一个趋势中N个中枢对应的压制均线大致相同**——如第一中枢被89日均线压制，后续同趋势中枢很可能也被同一均线压制。
- 均线系统设置**必须根据实际走势来**，使均线系统与实际已有走势相吻合。
- 推荐均线参数：**5、13、21、34、55、89、144、233**（斐波那契序列）。

**板块轮动方法**：
- 8条均线将所有品种分为9类（最差=完全在所有均线下，最强=突破所有均线）。分类原则：本次反弹目前为止未曾攻克的最小周期均线。
- 一旦一个趋势级别走势在N类上出现**顶背驰**→先出来，有几天时间去找**已调整可再启动的品种或补涨品种**。
- **缠中说禅板块强弱指标**=板块内所有品种类别数的平均值，越大越强。

**节奏是永远的主题**：高手还是低手，最终考验的就是节奏。轮动只是节奏的一种方式，最重要的节奏还是**买卖点**。

### 关键定义

**缠中说禅板块强弱指标**：对一板块内所有品种，按均线系统分为9类（从最弱0类到最强9类），取所有品种类别数的算术平均值。该值越大，板块越强。

**均线分类法**：以反弹为例，8条均线（5、13、21、34、55、89、144、233）将品种分为9类：
- 第1类（最弱）：完全在所有均线下
- 第2类：站上5均线但未攻克13均线
- ……
- 第9类（最强）：站上所有均线

### 实操要点（镑美适用）

1. **均线系统设置**：在GBP/USD的H1或H4图上设置5、13、21、34、55、89、144、233周期均线，检验该系统是否与近期走势吻合（关键转折点是否与某条均线高度契合）。如果吻合度高，持续沿用。
2. **均线作为级别辅助判断**：如果反弹连5均线都摸不到→笔级别都不是。如果突破34均线但受制于55均线→级别有限，对应操作空间有限。
3. **品种对比（外汇应用）**：如果同时关注多个货币对（如GBP/USD、EUR/USD、AUD/USD），用均线分类法对比强弱，找到最强势的品种做多、最弱势的品种做空。
4. **顶分型+均线压力=风险第一**：当镑美在接近压力均线时出现短线顶背驰→风险第一，先出来。不因为"可能还涨"而赌。

## 第107课：如何操作短线反弹

### 核心观点

- 操作不能涉及任何预测，必须从**绝对性**出发。
- 对一个30分钟反弹，绝对能指出的是：至少有**一个30分钟级别的中枢**（由三个以上次级别走势类型重叠构成，即上下上的三个5分钟走势类型）。
- 没有绝对性保障上下上中最后一个上一定比第一个上有更高高点（特别是**奔走型反弹**，后上高点可能只刚好触及前上低点）。
- 因此**不能等上下上都完成才抛出**——必须从第一个上就开始分解操作。
- **统一处理方法**：30分钟级别反弹，按**5分钟的节奏**去处理（次级别分解）。
- 实际操作中，第一上与下出现后，可能走势形式就有了很大的绝对性确认。如：30分钟中枢后接第三类买点+非背驰力度强劲拉升→可以坐轿子等第二、三个中枢完成出现背驰后第三类卖点。
- **不要预测反弹还是反转**，没意义。反弹越搞越大自然成反转。

### 关键定义

**上升趋势形成的最精确定义**：在第一中枢后出现第三类买点并形成**非背驰类向上**。趋势形成后，只要趋势没有扭转信号，就可以持有。

**奔走型反弹**：上下上结构中，后一个上的高点可能只刚好触及前一个上的低点，使得整个中枢构造完成后获利空间极小。

### 实操要点（镑美适用）

1. **次级别分解操作法**：预期镑美出现M30级别反弹→按M5节奏操作。M5第一段上涨结束时先获利部分仓位，不等M30中枢完整形成。
2. **第一上结束后的分类**：
   - M5第一段上涨结束（出现M5顶背驰或顶分型）→减仓
   - M5回调后再上涨→观察是否突破前高、力度是否背驰
   - 如果第二段上涨力度强劲、非背驰→加仓/持有，等更大级别信号
3. **趋势确认标准**：M30第一中枢形成后，出现第三类买点（回调不进入中枢区间）+向上走势非背驰→确认M30上升趋势，可以持有。
4. **奔走型防范**：不假设反弹必然有可观空间，第一段上涨就开始兑现利润，以应对奔走型的可能。
5. **换品种轮动**（高级操作）：保持部分仓位在镑美，用余下仓位在其他货币对中寻找已调整到位的品种进行轮动，提高资金效率。

## 第108课：何谓底部？从月线看中期走势演化

### 核心观点

- **底部的精确定义**（走势类型角度）：第一类买点出现后，一直到该买点所引发的中枢第一次走出第三类买卖点前，都可以看成**底部构造的过程**。第三类卖点先出现=底部构造失败；第三类买点先出现=底部构造最终完成并展开新行情。顶部反过来。
- **底部的分型定义**（粗糙但实用）：底部=构成底分型的那个区间。跌破分型最低点=底部构成失败；有效站住分型区间上边沿=底部构造成功，至少展开一笔上行。
- **底部有级别**：日线底分型=日线级别底部，周线底分型=周线级别底部。
- **买入时机**：不是在底部区间上买，而是**在区间下探失败时买**（和中枢震荡操作一样），这才是最好的买点。
- 操作的基本原则：任何走势都逃不出**底—连接过程—顶**的节奏。底的操作=中枢震荡（先买后卖更安全），顶的操作=中枢震荡（先卖后买更安全），中间连接部分=持有。

### 关键定义

**底部（走势类型角度）**：第一类买点出现后→至该买点引发的中枢第一次出现第三类买卖点前的整个区间。
- 第三类买点先出现 → 底部构造成功，新行情展开
- 第三类卖点先出现 → 底部构造失败

**底部（分型角度）**：构成底分型的区间[最低点, 分型上边沿]。
- 跌破最低点 → 底部失败
- 有效站住上边沿 → 底部成功，至少展开一笔上行

### 实操要点（镑美适用）

1. **底部识别两种方法**：
   - **精确法**：镑美M30出现第一类买点（底背驰）后→等待中枢形成→观察第三类买卖点谁先出现→第三类买点=底部成功做多；第三类卖点=底部失败继续观望。
   - **粗糙法**：在H4或日线上识别底分型→标记底分型区间→跌破最低点=失败→站住上边沿=成功。
2. **底部区间的操作**：在底部构造过程中，按中枢震荡操作——**先买后卖**（区间下沿附近买入，区间上沿附近卖出），因为在底部构造中，先买后卖更安全（即使判断错误，下方空间有限）。
3. **级别选择**：
   - 日内交易者：用M30或H1的底分型区间作为底部判断
   - 波段交易者：用H4或日线的底分型区间
4. **下探失败=最佳买点**：当镑美在底分型区间内向下探底，但不破区间最低点且出现低级别底背驰时→这是最佳买入点，不是在区间上边沿追买。
5. **三阶段节奏**：
   - 底部构造阶段：先买后卖的中枢震荡操作
   - 上升连接阶段：持有，按操作级别的背驰信号卖出
   - 顶部构造阶段：先卖后买的中枢震荡操作
6. **底部构造失败的应对**：如果镑美底分型区间最低点被跌破→立即确认底部失败→等下一个底分型机会，不抱幻想。
