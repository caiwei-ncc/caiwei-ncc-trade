## 第19课：学习缠中说禅技术分析理论的关键

### 核心观点
- 缠论的整个推导过程与几何证明类似，需要严格的数学思维。
- **必须放下以前学过的一切技术分析方法**，因为缠论与所有曾有技术分析方法的根本思路不同。
- 传统技术分析都是从**神秘的先验前提**出发，把或然当必然。
- 缠论是从市场的"不患"出发——任何市场参与者都逃不掉的最基本规律。
- 技术分析的最终意义**不是去预测市场要干什么，而是市场正在干什么**，是一种**当下的直观**。

### 实操要点（镑美适用）
- 分析镑美走势时，不要套用波浪理论、江恩角度线等先验框架，只用缠论的中枢、买卖点体系。
- 不预测，只看当下走势结构。

## 第20课：缠中说禅走势中枢级别扩张及第三类买卖点

### ★★★ 中枢的两种后续走势
1. **中枢延伸**：围绕中枢产生的前后次级别波动都至少有一个触及中枢区间。
2. **产生新的同级别中枢**：在趋势中，同级别前后中枢不能有任何重叠。

### ★★★ 走势中枢级别扩展
如果围绕新中枢产生的波动触及了前面中枢延续时的某个瞬间波动区间，则**不能认为是趋势**，而是产生一个**更大级别的中枢**。

### ★★★ 走势级别延续定理

| 编号 | 定理 |
|------|------|
| **定理一** | 在更大级别中枢产生前，该级别走势类型将延续 |
| **定理二** | 更大级别中枢产生，当且仅当围绕连续两个同级别中枢产生的波动区间产生重叠 |

### ★★★ 中枢的数学表达与Z走势段

**Z走势段定义**：与中枢形成方向一致的次级别走势类型。

**四个关键指标**：
| 指标 | 定义 |
|------|------|
| **GG** | max(gn)，所有Z走势段高点中的最高值 |
| **G** | min(gn)，所有Z走势段高点中的最低值 |
| **D** | max(dn)，所有Z走势段低点中的最高值 |
| **DD** | min(dn)，所有Z走势段低点中的最低值 |
| **ZG** | min(g1, g2)，中枢区间上沿 |
| **ZD** | max(d1, d2)，中枢区间下沿 |

### ★★★ 走势中枢中心定理

| 编号 | 定理 |
|------|------|
| **中心定理一** | 中枢延伸等价于任意区间[dn, gn]与[ZD, ZG]有重叠。若有Zn使得 dn＞ZG 或 gn＜ZD，则必然产生高级别中枢或趋势及延续 |
| **中心定理二** | 前后同级别两个中枢：后GG＜前DD → 下跌及延续；后DD＞前GG → 上涨及延续 |

### ★★★ 第三类买卖点定理

| 类型 | 精确定义 |
|------|----------|
| **第三类买点** | 一个次级别走势类型**向上离开**中枢，然后以一个次级别走势类型**回试**，其**低点不跌破ZG** |
| **第三类卖点** | 一个次级别走势类型**向下离开**中枢，然后以一个次级别走势类型**回抽**，其**高点不升破ZD** |

### 第三类买卖点的关键注意事项
1. 比第一、二类要**后知后觉**，但适合短线。
2. **必须是第一次**回调/回抽不回到中枢内才算。
3. 第三类买卖点后**不必然是趋势**，也可能进入更大级别的盘整。

### 实操要点（镑美适用）
- 在1H图上识别中枢[ZD, ZG]后，观察15M级别走势是否向上/向下离开中枢。
- 离开后的回试/回抽**不回到中枢内** → 第三类买卖点 → 入场信号。
- **ZG是做多方向的关键支撑，ZD是做空方向的关键阻力**。

## 第21课：缠中说禅买卖点分析的完备性

### ★★★ 核心定理：买卖点的完备性

> **市场必然产生赢利的买卖点，只有第一、二、三类。**

### ★★★ 三类买点之间的关系

| 关系 | 结论 |
|------|------|
| 第一类 vs 第二类 | **不可能重合** |
| 第一类 vs 第三类 | **不可能重合** |
| 第二类 vs 第三类 | **可能重合** |

**第二类与第三类买点重合条件**：第一类买点出现后，一个次级别走势凌厉地直接上破前面下跌的最后一个中枢，然后回抽不触及该中枢。

**实际意义**：一旦出现第二、三类买点重合，大级别上涨往往随之出现。

### ★★★ 升跌完备性定理

> **市场中的任何向上与下跌，都必然从三类买卖点中的某一类开始以及结束。**

### 实操要点（镑美适用）
- 任何不属于三类买卖点的入场都是"赌"。
- 第二类与第三类买点重合时信号特别强烈，可考虑加大仓位。

## 第24课：MACD对背弛的辅助判断

### ★★★ 背驰-买卖点定理

> **任一背驰都必然制造某级别的买卖点，任一级别的买卖点都必然源自某级别走势的背驰。**

### ★★★ 用MACD判断背驰的标准方法

**三段结构**：A段（第一段趋势）→ B段（中间中枢，MACD黄白线回拉0轴附近）→ C段（第二段同向趋势）

**判断标准**：C段MACD柱子面积比A段小 → 背驰成立。

**判断技巧**：柱子伸长力度变慢时，把已出现的面积乘以2即可当成该段面积 → 可在最高/最低价位附近判断。

### ★★★ 盘整背驰判断

**C段不破中枢**：MACD柱子C段面积小于A段 → 必有回跌。

**C段上破中枢但MACD面积小于A段**：
- 回跌**不重新跌回中枢** → 次级别第一类买点回补 → 构成该级别的**第三类买点**。
- 回跌**重新跌回中枢** → 继续盘整。

### ★★★ 背驰 vs 盘整背驰的区别

| 项目 | 趋势背驰 | 盘整背驰 |
|------|---------|---------|
| 位置 | 至少第二个中枢之后 | 第一个中枢 |
| 回跌力度 | **至少**重新回到B段中枢里 | 不一定回到中枢 |
| 精确性 | 结合中枢分析100%绝对 | 同样绝对，但需区分两种后续情况 |

### 实操要点（镑美适用）
- **这是我们系统中"15M背驰确认"的核心工具**。
- 在15M图上识别A、B、C三段，比较C段与A段的MACD柱子面积。
- C段面积小于A段 → 背驰/盘整背驰成立 → 入场信号。
- 盘整背驰后如果不回到中枢 → 可能是第三类买点 → 加仓机会。

## 第25课：吻，MACD、背弛、中枢

### 核心澄清
- **"吻"是均线系统的概念，与中枢是完全不同的概念**，不能混在一起。
- 均线系统和MACD等指标一样，只是辅助性工具。
- 中枢是严格定义的、绝对的。

### MACD参数与灵敏度
- 一般参数取12、26、9。
- 太快速的走势需用**MACD柱子伸长高度**来判断乖离极限。

### ★★★ 标准两个中枢上涨的MACD形态
1. MACD黄白线从0轴下上穿 → 第一个中枢形成 → 第二类买点。
2. 突破第一个中枢 → MACD快速拉起 → **最有力度的一段**。
3. 黄白线回到0轴附近 → 第二个中枢形成。
4. 突破第二个中枢 → MACD不创新高/面积缩小 → 背驰 → 上涨结束。

### 背驰回跌力度与级别的关系（极其重要）
- 日线走势刚开始，MACD刚创新高 → 5分钟背驰下跌力度有限。
- **日线走势最后阶段** → 1分钟背驰就足以引发暴跌。
- **必须多级别综合考察**。

### 实操要点（镑美适用）
- **4H MACD黄白线在0轴上方** → 做多；**在0轴下方** → 做空。这就是"防狼术前置过滤"的理论依据。
- 标准两个中枢上涨的MACD形态 = 判断走势处于哪个阶段的核心方法。

## 第27课：盘整背驰与历史性底部

### ★★★ 趋势背驰的统计规律
- **第二个中枢后就产生背驰的情况占绝大多数**（日线以上级别达90%以上）。

### ★★★ 背驰段的定义

> **在某级别的某类型走势中，如果构成背驰或盘整背驰，就把这段走势类型称为某级别的背驰段。**

### ★★★ 区间套定理——精确大转折点寻找程序

> **某大级别的转折点，可以通过不同级别背驰段的逐级收缩范围而确定。**

**具体方法**：大级别找背驰段 → 次级别找该背驰段内的背驰段 → 反复进行 → 最终在最低级别背驰段确定的范围内找到转折点。

### 实操要点（镑美适用）
- **区间套是我们系统"15M定入场"的理论基础**：4H找大方向 → 1H找背驰段 → 15M精确定位。
- 4H级别第二个中枢后就要密切注意背驰出现。

## 第29课：转折的力度与级别

### ★★★ 背驰-转折定理

> **某级别趋势的背驰将导致该趋势最后一个中枢的级别扩展、该级别更大级别的盘整或该级别以上级别的反趋势。**

### ★★★ 背驰后的三种情况

| 情况 | 描述 | 力度 |
|------|------|------|
| **第一种** | 反弹不能回到最后中枢内 → 中枢级别扩展 | 最弱 |
| **第二种** | 反弹回到中枢内 → 更大级别盘整 | 中等转折 |
| **第三种** | 反弹回到中枢并继续发展 → 反趋势 | 最强转折 |

### 如何区分
1. 第一波次级别反弹是否回到最后中枢内？不能 → 第一种；能 → 第二或第三种。
2. 突破中枢后是否形成第三类买点？形成 → 继续持有；不形成 → 退出。

### 实操要点（镑美适用）
- 15M入场后观察是否能回到1H最后一个中枢里。
- 不能回到 → 反弹力度可疑 → 考虑减仓。
- 能回到 → 转折确认 → 持仓等第三类买点。

## 第30课：缠中说禅理论的绝对性

### ★★★ 理论的基础前提
- 必须是**价格充分有效市场**。
- 必须是**非完全绝对趋同交易**。

> **只要满足这两个前提，本ID的理论就永远绝对有效。**

### 理论的当下性
- 所有判断都建立在**当下**之上。
- 走势是当下构成中的，判断同样是在当下构成的。

### 实操要点（镑美适用）
- 镑美是价格充分有效市场中的非完全绝对趋同交易 → 缠论完全适用。
- 所有分析必须基于当下走势，不预测。

## 第31课：资金管理的最稳固基础

### ★★★ 资金管理的核心原则
1. 资金必须长期无压力，不能有任何借贷。
2. 自己的资金一定不能交给别人管理。
3. 不能把自己放置在危险的境地。

### 操作中最坏的习惯
- 股票不断上涨就不断加仓 → 一定出问题。
- **宁愿不断跌不断买，也绝对不往上加码。**

### 实操要点（镑美适用）
- 每笔交易手数开仓前确定好，开仓后**不加仓**（除非满足加仓触发条件）。
- 加仓的前提：前序仓位止损已移至成本价（保本）。

## 第32课：走势的当下与投资者的思维方式

### ★★★ 当下性原则
- **所有在市场上的错误都是离开当下的直观，用想象、用情绪来代替。**

### ★★★ 走势的当下判断方法
1. A走出来后**不能预测一定有B**。
2. b段是否背驰由b段**当下走势**决定。
3. 参考次级别图看当下结构。
4. **一切都是当下的，不需要预测。**

### 实操要点（镑美适用）
- 不要预设镑美今天一定涨或跌，看当下走势结构。
- 持仓管理基于当下走势：15M出现背驰就减仓，第三类买点出现就加仓。

## 第33课：走势的多义性

### ★★★ 三种多义性
1. **中枢延伸造成的**：中枢延伸不超过5段，6段以上 → 构成更大级别中枢。
2. **模本简略造成的**：不同级别图是不同精度的模本。
3. **走势分析中的多种合理释义**（有实质意义）。

### ★★★ 纯中枢角度对背驰的释义
- **顶背驰** = 最后中枢向上离开比向下离开要弱 → 向下离开都能拉回 → 向上离开当然也能拉回。

### 围绕中枢的操作原则（总结）
1. 向下离开中枢出现**底背驰** → 可以介入。
2. 回拉出现**顶背驰**且不能超越前一个高点 → 一定要走。
3. 能超越且次级别回抽不回中枢 → 第三类买点 → 买回来。
4. 从底背驰开始的回拉不能回到中枢 → 第三类卖点 → 必须走。

### 实操要点（镑美适用）
- 1H中枢识别后，围绕中枢操作完全按上述原则。
- 中枢延伸超过5段 → 注意可能形成更大级别中枢。

## 第35课：给基础差的同学补补课

### ★★★ 最低级别的严格递归定义
1. 最低级别：每笔交易。连续三笔相同价位 → 最低级别中枢。
2. 逐级向上递归：三个连续低级别走势类型的重叠 → 高一级别中枢。

### ★★★ 走势分解定理
- **定理一**：任何走势可分解为同级别盘整、下跌、上涨三种走势类型的连接。
- **定理二**：任何走势类型至少由**三段以上次级别走势类型**构成。

### ★★★ 买卖点级别定理

> **大级别的买卖点必然是次级别以下某一级别的买卖点。**

### 实操要点（镑美适用）
- 15M的背驰有可能刚好是1H甚至4H的买卖点 → 区间套确认时信号最强。

## 第36课：走势类型连接结合性的简单运用

### ★★★ 结合律
**A+B+C = (A+B)+C = A+(B+C)**，级别可以不同。

### ★★★ 四种应用
1. **级别分解**：同一段走势可按不同级别分解。
2. **当下判断**：不同分解下看到不同含义。
3. **走势重新组合**：尽量避免中枢扩展。
4. **中枢震荡中的应用**：围绕大中枢的次级别第三类买卖点可能提前出现。

### 实操要点（镑美适用）
- 在1H中枢震荡中，留意15M级别的第三类买卖点 → 可能提前于1H级别信号出现。
- 组合原则：**避繁就简** → 选择使中枢结构最清晰的组合方式。

## 附录：关键操作判断标准速查

### MACD判断背驰的标准流程（第24课）
1. 识别A段（第一段趋势方向走势）
2. 识别B段（中间中枢，MACD黄白线回拉0轴附近）
3. 识别C段（第二段同向走势）
4. 比较C段与A段的MACD柱子面积
5. C < A → 背驰成立

### 第三类买卖点判断流程（第20课）
1. 识别中枢[ZD, ZG]
2. 观察次级别走势是否向上/向下离开中枢
3. 观察其后的次级别回试/回抽
4. 回试低点不跌破ZG → 第三类买点
5. 回抽高点不升破ZD → 第三类卖点
6. 必须是第一次

### 区间套精确定位流程（第27课）
1. 大级别找到背驰段
2. 次级别找该背驰段内的背驰段
3. 反复进行直到最低操作级别
4. 最终在最低级别背驰段确定的范围内找到转折点

### 背驰后三种情况判断流程（第29课）
1. 趋势背驰后，观察第一波次级别反弹
2. 不能回到最后中枢内 → 最弱（中枢扩展）→ 考虑快速退出
3. 回到中枢内 → 转折确认 → 继续观察
4. 突破中枢后回试不回中枢 → 第三类买点 → 继续持有/加仓
5. 回试回到中枢 → 盘整 → 考虑退出

### 标准两中枢上涨MACD形态（第25课）
1. MACD黄白线从0轴下上穿 → 第一个中枢形成 → 第二类买点
2. 突破第一中枢 → MACD快速拉起 → 最有力度的一段
3. 黄白线回到0轴附近 → 第二个中枢形成
4. 突破第二中枢 → MACD不创新高/面积缩小 → 背驰 → 上涨结束
